# Employee Data Management

## Setup Steps
1. Clone the repository: `git clone https://github.com/syaifulmain/employee-data-management.git`
2. Enter the project directory: `cd employee-data-management`
3. Install dependencies: `npm install`
4. Copy the example env: `cp .env.example .env`
5. Adjust environment variables in `.env` as needed.

## Running the Project
- Production build: `npm run build`
- Development mode: `npm run dev`
- Run the build: `npm start`
